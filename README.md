# Clack Growth Company

Site institucional one-page da Clack Growth Company, construída em Next.js com foco em apresentação comercial, método proprietário e geração de autoridade.

## Seções

- Hero com CTA duplo
- O novo cenário
- Quem somos
- Método Growth 360°
- Como o método funciona
- O que entregamos
- Especialidades
- Resultados
- Por que o método funciona
- Plano de crescimento
- Experiência
- A jornada
- Manifesto

## Stack

- Next.js
- React
- TypeScript
- CSS global

## Como executar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Publicação na Vercel

1. Conecte este repositório ao projeto na Vercel.
2. Faça deploy de produção.
3. A Vercel vai publicar a URL pública automaticamente.

## Observação

O layout foi pensado para uma experiência de alto impacto visual, com fundo escuro, identidade roxo/laranja e foco no Método Growth 360°.
